## How to Run

1. Clone repo
2. run `npm install discord.js` and `npm install pg`
3. Create .env - `cp .env.example .env`
4. Fill in discord bot token, bot id, and server id
5. Deploy commands - `node deploy-commands.js`
6. Start bot - `node index.js`
